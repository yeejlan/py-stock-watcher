py-stock-watcher
---

基于Python3, PyQt5写的股票行情小工具

安装，设置，运行:

1. 需要[Python3](https://www.python.org/download)和[PyQt6](https://riverbankcomputing.com/software/pyqt/download)


2. 打开config.ini设置需要显示的股票数据，目前支持沪深股票，港股，国内期货行情

3. 运行main.py, 双击行情可以显示分时图
